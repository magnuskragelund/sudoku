## Flow Summary

### HOST Powers
1. **Start New Rounds** - Can generate new puzzles for all players at any time (header + modals)
2. **Control Game Flow** - Can pause/resume for everyone (shared with players)
3. **Exit Anytime** - Can leave session via header button or modals (doesn't end session for others)

### PLAYER Powers
1. **Control Game Flow** - Can pause/resume for everyone (shared control)
2. **Continue Playing** - Can keep solving even after someone wins
3. **Exit Safely** - Can leave session via pause menu or modal buttons only (no accidental exits)

### Key Behaviors
- **Pause is Shared**: Any player or host can pause, and it pauses for everyone
- **Resume is Shared**: Any player or host can resume, and it resumes for everyone
- **Lives are Individual**: Each player has their own lives
- **Completion is Broadcast**: When anyone completes, all players are notified
- **New Round (Host Only)**: Only host can start a fresh puzzle for everyone
- **Player Exit Safety**: Players have no header button (prevents accidental exits); must use pause menu or modals
- **Consistent Exit Labels**: All multiplayer exit buttons say "Leave Current Game" (not "End Game" or "New Game")
